## BuildingTowersEasy

div1 250

どの位置に最も高い建物を置くかを決め打ちする。

ある座標`i`に決めたとして、各`t`について座標の差の絶対値分だけ高さが違うことになる。

つまり`| i - x[j] |`だけ高さを変えられる。

`t[j] + |i - x[j]|`の中で最小値を探せば良い。

x座標0の位置には置けないことに注意。
